📁

프로젝트 시작하기

홈 화면에서 '새 프로젝트 선택'을 눌러 관리하고 싶은 폴더를 선택하세요.
DGit 저장소가 없는 폴더는 자동으로 초기화할 수 있습니다.

💾

변경사항 커밋하기

파일을 수정하면 '프로젝트 파일' 목록에서 상태가 'modified'로 바뀝니다.
'변경사항 커밋' 버튼을 눌러 변경된 파일을 저장하세요.

↩️

이전 버전으로 복원

'커밋 히스토리' 탭에서 특정 커밋을 선택하고 '복원' 버튼을 누르면 해당 시점으로 돌아갈 수 있습니다.

⚡

DGit 명령어 소개

dgit init

새로운 DGit 저장소를 현재 디렉터리에 생성합니다.

dgit status

현재 프로젝트의 변경된 파일 상태를 확인합니다.

dgit add [파일/폴더]

변경사항을 스테이징 영역에 추가합니다.

dgit commit -m "[메시지]"

스테이징된 파일을 커밋하여 저장소에 기록합니다.

dgit log

커밋 히스토리를 시간 순서대로 보여줍니다.
